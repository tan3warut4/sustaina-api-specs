Position Search

Search Position

Return positions in a tree structure by tenant.

  • Default: return matched node(s) and full descendants.
  • Set expand.ancestors=true to include the ancestor path to the real root.
  • We never mutate parentId; deep nodes are rendered as top-level entries when their parents are not included.
Authorizations:
bearerAuth
Request Body schema: application/json
required
required
Array of objects

List of AND conditions. Supported fields include: name, code, type, isActive, createdAt, updatedAt, startTerm, endTerm, tags, id, parentId, sortingLevel, etc.

object
Default: {}

Control tree expansion. descendants defines children depth (ALL or 0..N). ancestors includes parent chain to root. rootMode is reserved for future use; currently parentId is never mutated.
